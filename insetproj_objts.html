India Set: Objects

India Set:Objects is a set of objects that is intended to compliment the rest of the set.


The proposed set will contain objects that can be used with other newgrfs in this set or others of one's choosing to decorate and create one's imaginary world. Let your imagination flourish!


To contribute to this section of the India Set , kindly refer to the the Contributions tab.
